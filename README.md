Glow Haven is a sophisticated, full-stack e-commerce platform dedicated to beauty and skincare products. Built with a focus on elegant UI/UX and seamless performance, it provides a complete shopping experience from product discovery to a dynamic checkout process.

üöÄ Key Features
Modern Aesthetic UI: A minimalist, "clean beauty" inspired design with a responsive layout for mobile and desktop.

Dynamic AJAX Shopping Cart: Users can add items to their cart and manage quantities in real-time without page refreshes.

Real-time Cart Feedback: A visual notification system (badge) that tracks the number of items added.

Session-Persistent Storage: Uses PHP sessions to ensure the user's cart remains intact while browsing different sections.

Product Catalog: A structured display of skincare and haircare categories fetched dynamically.

Smooth Transitions: Enhanced user experience using CSS animations and Vanilla JavaScript.
